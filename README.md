# ⚽ StatsBomb Football Analytics

Data-driven insights into football performance using StatsBomb's open data.

## 📊 Project Overview

This project analyzes football match data from StatsBomb to uncover tactical patterns, player performance metrics, and team dynamics.

## 🛠️ Technologies

- Python 3.x
- Pandas, NumPy
- Matplotlib, Seaborn, Plotly
- StatsBombPy
- Jupyter Notebook

## 🚀 Getting Started

1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Launch Jupyter: `jupyter notebook`
4. Open notebooks in the `notebooks/` folder

## 📁 Project Structure
```
├── data/          # StatsBomb data
├── notebooks/     # Analysis notebooks
├── src/           # Reusable Python functions
└── outputs/       # Visualizations and reports
```
